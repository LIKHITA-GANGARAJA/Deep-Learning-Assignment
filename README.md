This repository contains two deep learning implementations developed for my course assignment.
The CNN model performs image classification using the Animals-10 dataset, and the RNN model uses an LSTM-based architecture for sentiment analysis on the Twitter US Airline Sentiment dataset. Both projects include data preprocessing, model building, training, evaluation, and the required Google Colab notebooks with PDF exports.

Datasets Used
 Animals-10 (CNN): https://www.kaggle.com/datasets/alessiocorrado99/animals10
 Twitter US Airline Sentiment (RNN): https://www.kaggle.com/datasets/crowdflower/twitter-airline-sentiment
